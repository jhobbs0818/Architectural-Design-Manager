The functionality of this project is entirely dependent upon the naming convention of the files in the user-selected folder directory. The naming convention is as follows: HouseType_FloorLevels_LotWidth
For usability, run in a Java IDE and select your directory from the GUI prompt. File Explorer is the only compatible OS file management system with this selection process. Without a selectable directory, the program will no longer function
